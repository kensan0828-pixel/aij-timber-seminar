## アシスト方針

1. **技術的正確さを最優先**とし、規準の文言・式・記号は原典に忠実に扱う
2. スライド資料は**旧基準との対比を明確に**示す（表形式・Before/After形式を積極活用）
3. 構造設計者向けのため、**専門用語は略さず使用**してよい
4. **1時間・2名登壇・質疑込み**のペース感を常に意識する
5. 変更点のみを重点説明し、**「変更なし」の項目は列挙しない**
6. 変更種別は `[追加]` `[削除]` `[移動]` `[改定]` で明示する


### 数式記法のルール

| 用途 | 記法 | 備考 |
|------|------|------|
| インライン数式 | `$...$` | 本文中の変数・記号すべて |
| 独立数式ブロック | `$$...$$` | 1スライドに1式まで |
| コードブロック内 | LaTeX使用禁止 | 代わりにテキスト＋インライン数式 |
| `\text{}` 内の日本語 | 使用禁止 | 日本語は数式外に出す |

### 主なLaTeX変換対応表

| プレーンテキスト | LaTeX |
|---|---|
| `_mF`、`_nF`、`_0F` | `$_mF$`、`$_nF$`、`$_0F$` |
| `E_b0.05` | `$E_{b0.05}$` |
| `C_k`、`C_s`、`C_b` | `$C_k$`、`$C_s$`、`$C_b$` |
| `F_k`、`F_c`、`F_kp` | `$F_k$`、`$F_c$`、`$F_{kp}$` |
| `_jK_r`、`_jK_d`、`_jK_m` | `$_jK_r$`、`$_jK_d$`、`$_jK_m$` |
| `R_f1`、`R_f2` | `$R_{f1}$`、`$R_{f2}$` |
| `F_e//`、`F_e⊥` | `$F_{e\parallel}$`、`$F_{e\perp}$` |
| `E₀`、`G₀` | `$E_0$`、`$G_0$` |
| `h₀`、`e₂`、`k₀` | `$h_0$`、`$e_2$`、`$k_0$` |
