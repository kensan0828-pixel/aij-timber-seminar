## 5. Marpスライドの仕様（確定済み）

### フロントマター（全ファイル共通）

```yaml
---
marp: true
theme: default
size: 16:9
paginate: true
math: mathjax
style: |
  section {
    font-size: 20px;
  }
  h1 {
    font-size: 36px;
  }
  h2 {
    font-size: 27px;
  }
---
```

### レイアウトルール

- 1枚あたりの項目数：**3〜5項目**を厳守
- 箇条書きの階層：**2段まで**
- 表の列数：**最大5列**
- 数式：**1スライド1式**を基本
- 文字数の目安：1項目あたり全角**40字以内**

### 表記ルール

- 節番号は規準体系に準拠（例：`## 602.3 ドリフトピン接合`）
- 変更種別を角括弧で付記：`[改定]`・`[追加]`・`[削除]`・`[移動]`
- 本文改定は `[本]`、解説改定は `[解]` で区別
- 数式はLaTeX記法（`$...$` / `$$...$$`）を使用
  - インライン数式：`$...$`
  - 独立数式ブロック：`$$...$$`
  - コードブロック（`` ` ``）内はLaTeXが効かないため使用しない
  - `\text{}` 内の日本語はMathJaxでは一応可だが、なるべく数式外に出す

### Before/After比較の書き方

- 旧基準との対比は**必ず明示**し、表形式（旧・新・備考の3列）を優先
- 数値・係数・式が変わった場合は旧値と新値を**並列で示す**

## 6. 用語・略語の統一表

| 略語 | 意味 |
|------|------|
| `[本]` | 本文の改定 |
| `[解]` | 解説の改定 |
| $R_{f1}$ | 地震荷重割増率 |
| $R_{f2}$ | 地震時応力割増率 |
| JA〜JC | 接合種別（靭性クラス） |
| CLT | 直交集成板（Cross Laminated Timber） |
| LVL | 単板積層材（Laminated Veneer Lumber） |
| $E_{b0.05}$ | 曲げの基準弾性係数の95%下側許容限界値 |
| $M_p$ | 全塑性曲げモーメント |
| $_jK_r$ | 靭性係数（2026年版より削除） |

1. `seminar_slides.md` で対象章の改定内容を確認（変更点のみ抽出）
